# Code-Server-Web

项目为本人本科毕业设计

配合BUCTOJ使用，提供一个可以线上使用vscode完成代码编辑调试和提交的平台

前端采用Vue.js（v3.5.13）作为UI框架，采用primevue、antdvue组件库，使用pinia作为状态管理

后端采用Go编写，选用gin作为服务器框架，gorm连接数据库，使用mysql作为数据库
